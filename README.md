Synapse AI: AI-Powered Study & Content Automation Platform

Synapse AI is an AI-powered study and content automation platform built with a modern web stack to improve learning efficiency and user engagement. The platform generates flashcards, quizzes, and summaries using AI, demonstrating how data-driven automation, cloud infrastructure, and product-focused design come together in modern SaaS-style applications.

Features

AI-Driven Content Generation: Uses the Gemini AI API to automatically generate flashcards, quizzes, and summaries, reducing manual content creation time by up to 70%.

User Engagement Focus: Designed to support quick, structured learning experiences that improve usability and retention.

Automated Workflows: Eliminates repetitive study preparation tasks, allowing users to focus on outcomes rather than manual effort.

Cloud-Based Architecture: Built on Firebase services for real-time data synchronization, scalability, and low-latency performance.

Secure Authentication: Implements Firebase Authentication for secure user access and session management.

Responsive Web Interface: Clean, modern UI built with JavaScript and Tailwind CSS, optimized for both desktop and mobile devices.

Tech Stack

Frontend: JavaScript, Tailwind CSS
Backend / Cloud: Firebase Authentication, Firestore, Gemini API

Getting Started

Clone the repository and install dependencies:
git clone https://github.com/Pranavrai207/Synapse-AI.git

cd Synapse-AI
npm install

Set up Firebase Authentication and Firestore in a Firebase project, and configure Gemini API access using your API key.

Start the application:
npm start

Open the local development URL shown in the terminal to access the Synapse AI interface.
